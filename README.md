## 简介

一个基于 `vue2`,`@vue/cli`,`vuetify`的简易个人博客。

## 特性

- 较旧的技术
- 勉强的屏幕适配
- 看不清的黑夜模式
- 毫无特色还是借鉴的博客风格

可以说毫无特性了。

## 安装

- 获取项目源码
- 安装依赖
  ```bash
  yarn install
  ```
- 运行
  ```bash
    yarn dev
  ```
- 打包
  ```bash
    yarn build
  ```

## 技术栈

- [`Vue2`](https://cn.vuejs.org/)
- [Vue2 的全家桶](https://cn.vuejs.org/)
